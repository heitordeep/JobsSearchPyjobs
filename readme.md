# Search for jobs on the Pyjobs website

## Before you start:

You need the version ```Python 3.7```

1) See available commands:
   ```shell
    $ make help
   ```

1) Installation requirements:
   ```shell
    $ make requirements
   ```

## Run application:
   ```shell
    $ make run
   ```

   The data will be stored in sqlite.
